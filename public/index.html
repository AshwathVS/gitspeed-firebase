itspeed

      - the chrome extension for programmers

Instructions

  • Install the extension using chrome store.

  • Login using github OAuth through the extension.

  • Before you start to commit files, you will need to add paths which can be done using the + icon.

  • Once that is done, you are good to go. Use to plugin to commit all the programs that you want to keep track of.

  • Even though the plugin is aimed towards competitive programmers who miss out on keeping track of the programs that they code at various sites, the plugin can also be used for note taking, saving up interesting code too.

Note

  • Your data is secured with firestore. If you choose to opt out, make sure to hit the delete data icon so that we delete all your files stored in local storage and the cloud.
